Inställningar för Integritet och säkerhet

Avsnitten som listas nedan beskriver inställningarna för Integritet och säkerhet. För att se inställningspanelerna, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Klicka på kategorin Integritet och säkerhet. Om inga underkategorier är synliga, dubbelklicka på Integritet och säkerhet för att veckla upp listan och klicka sedan på namnet på de inställningar du vill visa eller ändra.

För hjälp med en panels inställningar, klicka på lämplig länk nedan.

Tillhörande avsnitt:
  • Kakor
  • Bilder
  • Lösenord
  • Huvudlösenord
  • SSL
  • Certifikat
  • Validering
